# Server Room VR

## Execução pela Unity

1. Ter a unity versão 2022.2.6f1
2. Com o Quest 2 ou 3, baixar e instalar o sofware **Meta Quest Link** e fazer o **Quest Link** com o PC.
3. Com o Quest conectado basta executar o programa.
